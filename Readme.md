## Basic Setup and Node Js Info

### NPM  -  Used to Install Node Packages 

- To Initilize as npm project

```bash
npm init 
```
- To Install the package like Angular Cli and Express
```bash
npm i @angular/cli
```

```bash
npm i express
```

### package.JSON 
- It defines the packages/dependencies we installed

### package-lock.json 
-  It defines the tree of packages/dependencies we have installed i.e. like we have installed @angularCli which need some other packages to run itself will be defined in this file
